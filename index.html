Google Sheets 연동 설정

Apps Script 웹 앱을 배포한 후 URL을 입력하면 Sheets와 양방향 연동됩니다.

  1. Google Sheets > 확장 프로그램 > Apps Script
  2. Code.gs에 제공된 코드 붙여넣기
  3. 배포 > 새 배포 > 유형: 웹 앱
    실행 계정: 나 / 액세스: 모든 사용자
  4. 배포 URL 복사 후 아래에 붙여넣기
나중에 설정 (기본 데이터로 계속)
메시지 추가
성과 지표 (선택)
링크 (선택)
스테이지 추가
새 단계가 Sheets에 추가됩니다

스테이지 자체는 Sheets에 별도 행을 만들지 않습니다. 첫 메시지를 추가하면 해당 스테이지 행이 생성됩니다.
